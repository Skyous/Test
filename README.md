## System Requirements
To be able to run Laravel Boilerplate you have to meet the following requirements:
- PHP > 7.1
- PHP Extensions: PDO, cURL, Mbstring, Tokenizer, Mcrypt, XML, GD
- Node.js > 6.0
- Composer > 1.0.0
